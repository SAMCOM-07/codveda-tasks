# Interactive Form Validation Project

A basic and responsive interactive form built using **HTML, CSS, and Vanilla JavaScript**.

This project demonstrates client-side form validation, real-time feedback, and basic user input handling.

---

## 🚀 Features

- Required field validation
- Email format validation
- Password length validation
- Confirm password matching
- Error message display
- Clean and responsive UI
- Prevents form submission when invalid

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)


### 2️⃣ Navigate into the project folder

```bash

```

### 3️⃣ Open in browser

Simply open the `index.html` file in your browser:

- Double-click the file  
OR  
- Right-click → Open with browser  

No additional dependencies required.

---

## 💡 How It Works

- JavaScript listens for form submission
- Prevents default submission behavior
- Validates each input field
- Displays error messages dynamically
- Only allows submission when all fields are valid
- Gives an alert if/when form is submitted successfully
